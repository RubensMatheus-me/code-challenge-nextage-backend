
# Task Manager

Backend - Desafio de Estágio

Este projeto representa a API Backend do desafio de estágio.
Foi desenvolvido com Spring Boot e tem como objetivo fornecer serviços de autenticação de usuários, gerenciamento de tarefas e recuperação de senha, garantindo segurança, persistência de dados e integração com o frontend.

## Funcionalidades

## ✅ Checklist de Funcionalidades

- [x] Criação de Usuários
- [x] Autenticação de Usuário (Login/JWT)
- [x] Recuperação de Senha via e-mail
- [x] Alteração de Senha com código de validação
- [x] Criação de Tarefas
- [x] Visualização de Tarefas
- [x] Edição de Tarefas
- [x] Exclusão de Tarefas
- [x] Filtros e Ordenação de Tarefas 
- [x] Criação de tarefas

## Tecnologias Utilizadas

Backend: Java 17, Spring Boot 3.x

Segurança: Spring Security, JWT

Banco de Dados: MySQL

ORM: Spring Data JPA / Hibernate

Gerenciamento de Dependências: Maven

Outras: Lombok e Validation API
